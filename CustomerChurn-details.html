Project Detail

Customer Churn Prediction is a project that aims to predict which customers will stop subscribing to a service or product, using historical data about customer behavior and characteristics, as well as their churn status. This project can help companies to identify the factors that influence churn, as well as design strategies to increase customer loyalty and retention.

There are three methods used in this project, namely:

  • Neural net: This method uses artificial neural networks to learn the patterns and relationships between input and output variables. Neural networks can handle complex and non-linear data, as well as have the ability to learn relevant features automatically.
  • Neural net + SMOTE: This method uses the same neural network as the previous method, but with the addition of the SMOTE technique. SMOTE is a technique to balance the imbalanced classes in the data, by creating synthetic samples from the minority class. Thus, the neural network can learn from more representative data and reduce the bias towards the majority class.
  • Neural net + SMOTE + Ensamble: This method uses the same neural network as the previous method, but with the addition of the ensamble technique. Ensamble is a technique to combine multiple machine learning models into one model that is more powerful and accurate. Thus, the neural network can reduce the variance and overfitting, as well as improve the generalization and robustness.

The following are the main steps of the project:

  1. Data Preprocessing: The data is cleaned and transformed, by dropping irrelevant or redundant columns, converting categorical variables into dummy variables, splitting the data into train and test sets, separating the target variable from the input variables, and standardizing the input variables.
  2. Model Neural Net: A neural network model is defined and trained, using four layers of neurons, batch normalization and dropout techniques, the 'binary_crossentropy' loss function, the 'RMSprop' optimizer, and the 'accuracy' metric.
  3. SMOTE: The SMOTE technique is applied to the train data, using the SMOTE function, with a sampling strategy of 0.46 and a number of nearest neighbors of 10.
  4. Ensamble: Two models are trained on the train data that has been oversampled by SMOTE: the same neural network model as before, and an XGBoost model. The two models are then used to make predictions on the test data, and the predictions are stacked together. A final model is created using the GradientBoostingClassifier function, and trained on the stacked predictions and the test output variable. The final model is then used to make predictions on the test data, using the stacked predictions as the input.

The following are the accuracy results of the three methods, using the precision, recall, f1-score, and support metrics:

  • Neural net: The neural network model achieved an accuracy of 0.81 on the test data, with a precision of 0.61, a recall of 0.58, and an f1-score of 0.60 for the positive class (churned customers). The model performed better on the negative class (non-churned customers), with a precision of 0.87, a recall of 0.88, and an f1-score of 0.87.
  • Neural net + SMOTE: The neural network model with SMOTE achieved an accuracy of 0.82 on the test data, with a precision of 0.61, a recall of 0.65, and an f1-score of 0.63 for the positive class. The model also performed better on the negative class, with a precision of 0.89, a recall of 0.87, and an f1-score of 0.88.
  • Neural net + SMOTE + Ensamble: The neural network model with SMOTE and ensamble achieved an accuracy of 0.81 on the test data, with a precision of 0.61, a recall of 0.63, and an f1-score of 0.62 for the positive class. The model also performed better on the negative class, with a precision of 0.88, a recall of 0.87, and an f1-score of 0.88.

From the results, it can be seen that the three methods have similar accuracy scores, but the neural network model with SMOTE has slightly higher recall and f1-score for the positive class, which means that it can identify more churned customers correctly. However, the neural network model with SMOTE and ensamble has slightly higher precision for the positive class, which means that it can reduce more false positives. Therefore, the best method depends on the trade-off between precision and recall, and the business objective of the project.
