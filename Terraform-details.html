Project Detail

This portfolio project showcases my skills and experience in creating a pipeline architecture using Terraform, Kubernetes, GitHub, and Dockerhub. The project aims to automate the creation of the pipeline infrastructure with the help of infrastructure as code. The project consists of several Terraform files that are responsible for creating the following resources:

  • A Dockerhub repository to store and distribute the docker images
  • A Kubernetes server with Oracle Cloud Infrastructure to run and coordinate the containerized applications
  • A GitHub repository with workflow actions to build, test, and deploy the applications
  • A reserved public IP address for the Kubernetes server

The project demonstrates my ability to use various tools and technologies, such as:

  • Terraform CLI and plugins to initialize, plan, apply, and destroy the infrastructure
  • OCI Terraform Provider to create and manage the OCI resources, such as VCN, subnet, instance, security list, etc.
  • Kubernetes Terraform Provider to create and manage the Kubernetes resources, such as cluster, node pool, service account, etc.
  • GitHub Terraform Provider to create and manage the GitHub resources, such as repository, workflow, and secret
  • Dockerhub Terraform Provider to create and manage the Dockerhub repository
