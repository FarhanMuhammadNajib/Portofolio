Rock Paper Scissors Image Recognition Detail

Rock paper scissors is a classic game that involves two players who make one of three possible hand gestures: rock (a closed fist), paper (a flat hand), or scissors (two fingers extended). The game is based on a simple rule: rock beats scissors, scissors beats paper, and paper beats rock.

Project Information

  1. Dataset: A Kaggle dataset of 2,188 images of rock paper scissors gestures
  2. Data Augmentation: Applied random transformations to the images to create new images
  3. Base Model: A pre-trained model called VGG19 that can recognize 1,000 different classes of objects
  4. Custom Classifier: A layer added on top of the base model that makes the final predictions for the three classes of gestures
  5. Transfer Learning: Fine-tuned the model to the dataset using feature extraction and fine-tuning strategies
  6. Callbacks: Used early stopping and reduce learning rate on plateau to monitor and improve the training process

Results

  • Achieved an accuracy of 99% on the validation set, and tested the model on some new images
  • Created a web app that allows the user to play rock paper scissors with the model using a webcam and a computer

Learning Outcomes

Learned how to use a pre-trained model, how to fine-tune it to a new task, how to apply data augmentation, and how to use callbacks to improve the training process
