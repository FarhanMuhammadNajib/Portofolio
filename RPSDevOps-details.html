Project Detail

I developed a web application that can recognize whether the image uploaded by the user is shaped like a rock, paper, or scissors. I used state-of-the-art deep learning techniques with the help of TensorFlow and Keras to create a model that can classify images with high accuracy. I also used Flask to create a web interface that is user-friendly and appealing.

I used Oracle Cloud Infrastructure as the provider of the server to run my web application. I chose Oracle Cloud because it offers high performance, security, and scalability. I can also manage my cloud resources easily and cost-effectively.

Key Points of My Project

  • I used GitHub Actions as a tool to perform CI/CD, which is an automated process to build, test, and deploy my web application to a Docker server.
  • I created a YAML file that defines the steps that GitHub Actions must perform to run my CI/CD pipeline. This file consists of two parts: Docker and Deploy.
    • The Docker part is responsible for logging in to Docker Hub, where I store the Docker image of my web application. Then, this part will build the Docker image from my source code and push it to Docker Hub.
    • The Deploy part depends on the Docker part and is responsible for logging into the Docker server with SSH, deleting the old image and container, running a new container from the latest image, and setting up the firewall to allow access to port 1000.
  • My CI/CD pipeline will run every time I do a push or pull request to the main branch on GitHub, or every time I trigger it manually from GitHub.
  • I used some useful Python libraries, such as PIL for processing images, TensorFlow and Keras for creating and training deep learning models, and Flask for creating web applications.

With this project, I demonstrate that I can develop a web application that uses deep learning and DevOps technologies. I hope this project can be my portfolio to showcase my skills in this field.
